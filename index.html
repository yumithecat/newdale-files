DamagePerms is a simple plugin to be able to control damage by using permissions! I recommend combining this with LuckPerms.

Tosplugin

This plugin is made by tvhee to support this network. The plugin works with a GUI where you HAVE to accept the TOS of the server. If accepted, they can play, if not they get kicked. The TOS configuration can be changed in the config.yml. Ask tvhee for support if you want a modified version.
